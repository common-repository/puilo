This plugin replaces the surname of Russian dictator Vladimir Putin in WordPress publications.

== Description ==

This plugin replaces the surname of Russian dictator Vladimir Putin in WordPress publications. 
In March 2014 this Ukrainian and Russian slogan was performed in its current form and became a nationwide hit in Ukraine during the events of the 2014 Russian military intervention in Ukraine.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/puilo directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Puilo Settings screen to configure the plugin.

== Frequently Asked Questions ==

Khuilo it is written in Wikipedia, is it the most similar to the Ukrainian word Хуйло.
